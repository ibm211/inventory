4 Caso 2 – EOQ: Análisis de costos de arena gruesa

4.1 Enunciado

  1. EOQ. La tonelada de arena gruesa se compra a $45 y se vende a un ritmo de 100 tn/semana. El proveedor siempre entrega a tiempo el pedido y el costo de mantener inventario es solamente el costo de oportunidad del dinero invertido que es de 0.5% semanal y el costo de ordenar es el flete por el cual paga $180.
  1. Analizar las variaciones de los costos semanales de ordenar y de mantener inventario para las siguientes cantidades de pedido: 50, 100, 200, 300, 400, 500 y 600.
  2. Representar gráficamente las características del modelo para Q= 50, 100, 200. Analizar el gráfico obtenido.
  3. ¿Cuál es la cantidad óptima?

4.2 Datos del problema

Variable Símbolo Valor Unidad
Costo unitario C 45 $/tonelada
Demanda d 100 toneladas/semana
Costo de oportunidad i 0.5% semanal
Costo de ordenar (flete) S 180 $/pedido

Nota: Dado que la demanda y los costos están en base semanal, trabajaremos en base semanal.

4.2.1 Costo de mantener por unidad por semana

H = C \times i = 45 \times 0.005 = 0.225 \text{ \$/tonelada-semana}

4.3 Modelo aplicable

Modelo EOQ en base semanal. Las fórmulas de costos semanales: \text{Costo semanal de ordenar} = \frac{d}{Q} \times S \text{Costo semanal de mantener} = \frac{Q}{2} \times H \text{Costo total semanal} = \frac{d}{Q} S + \frac{Q}{2} H Q^* = \sqrt{\frac{2dS}{H}}

4.4 Desarrollo

4.4.1 Resolución Matemática en LaTeX

a) Análisis de costos para diferentes cantidades de pedido

Para cada valor de Q se calcula:

Q = 50: \text{Costo ordenar} = \frac{100}{50} \times 180 = 360 \text{ \$/semana} \text{Costo mantener} = \frac{50}{2} \times 0.225 = 5.63 \text{ \$/semana}

Q = 100: \text{Costo ordenar} = \frac{100}{100} \times 180 = 180 \text{ \$/semana} \text{Costo mantener} = \frac{100}{2} \times 0.225 = 11.25 \text{ \$/semana}

Q = 200: \text{Costo ordenar} = \frac{100}{200} \times 180 = 90 \text{ \$/semana} \text{Costo mantener} = \frac{200}{2} \times 0.225 = 22.50 \text{ \$/semana}

Q = 300: \text{Costo ordenar} = \frac{100}{300} \times 180 = 60 \text{ \$/semana} \text{Costo mantener} = \frac{300}{2} \times 0.225 = 33.75 \text{ \$/semana}

Q = 400: \text{Costo ordenar} = \frac{100}{400} \times 180 = 45 \text{ \$/semana} \text{Costo mantener} = \frac{400}{2} \times 0.225 = 45 \text{ \$/semana}

Q = 500: \text{Costo ordenar} = \frac{100}{500} \times 180 = 36 \text{ \$/semana} \text{Costo mantener} = \frac{500}{2} \times 0.225 = 56.25 \text{ \$/semana}

Q = 600: \text{Costo ordenar} = \frac{100}{600} \times 180 = 30 \text{ \$/semana} \text{Costo mantener} = \frac{600}{2} \times 0.225 = 67.50 \text{ \$/semana}

Se observa que en Q = 400, el costo de ordenar ($45) es exactamente igual al costo de mantener ($45), lo que sugiere que el óptimo está en esa zona.

b) Representación gráfica IOH vs. Tiempo para Q = 50, 100, 200

Para cada valor de Q, la duración del ciclo es T = Q/d: - Q = 50: T = 50/100 = 0.5 semanas (3.5 días) - Q = 100: T = 100/100 = 1 semana - Q = 200: T = 200/100 = 2 semanas

Con Q más pequeño, los ciclos son más cortos y frecuentes, y el inventario promedio es menor, pero el costo de ordenar es mayor.

c) Cantidad óptima

Q^* = \sqrt{\frac{2dS}{H}} = \sqrt{\frac{2 \times 100 \times 180}{0.225}} = \sqrt{\frac{36{,}000}{0.225}} = \sqrt{160{,}000} = 400 \text{ toneladas} Verificación: TC^* = 2 \times 45 = 90 $/semana.

4.5 Implementación en R

Code 
# Variables
d <- 100
C <- 45
S <- 180
i <- 0.005
H <- C * i

# a. Análisis de variaciones
Qs <- c(50, 100, 200, 300, 400, 500, 600)
co <- (d / Qs) * S
cm <- (Qs / 2) * H
ct <- co + cm

resultados <- data.frame(Q = Qs, Costo_Ordenar = co, Costo_Mantener = cm, Costo_Total = ct)
print("Variaciones de costos semanales:")

Cantidad óptima: Q^* = 400 toneladas con un costo total semanal de $90.00.

4.7 Conclusiones

  1. Q* = 400 toneladas: Es la cantidad exacta que minimiza los costos semanales. En este punto, el costo de ordenar ($45/sem) iguala exactamente al costo de mantener ($45/sem).
  2. Sensibilidad a desviaciones: El costo total es relativamente insensible cerca del óptimo. Por ejemplo, pedir 300 o 500 toneladas genera un costo total de $93.75 y $92.25 respectivamente, solo 4-3% más que el óptimo. Sin embargo, cantidades muy alejadas como Q=50 generan un costo 4 veces mayor ($365.63/sem).
  3. Comportamiento de los costos: El costo de ordenar decrece hiperbólicamente con Q (a mayor pedido, menos fletes), mientras que el costo de mantener crece linealmente (más inventario promedio). La curva de costo total tiene forma de “U” con mínimo en Q*=400.
  4. Frecuencia óptima: Con Q*=400 toneladas y demanda de 100 ton/semana, se debe pedir cada 4 semanas (mensualmente), lo cual resulta práctico operativamente.
