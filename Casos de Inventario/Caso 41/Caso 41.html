42  Desarrollo del Caso 41

42.1 Enunciado literal

  1. EOQ. SARDIMAR S.A. es una empresa líder en la producción de atún y sardina… El encargado del área de operaciones cuenta con la siguiente información respecto a cada una de las etapas del proceso: Etapa Capacidad efectiva mensual (en miles de cajas) Merma ACONDICIONAMIENTO 1,000 15 % PREPARACIÓN 1,000 5 % ACABADO 800 2 % EMPAQUE 750 0 %

El producto se distribuye en cajas que contienen 50 latas.

El área de operaciones tarda 4 días en iniciar el proceso productivo. Se sabe que mantener una caja de conservas de atún almacenada ocasiona un costo anual del orden de los 10 soles, y el costo de arranque del proceso es de 2,000 soles. La empresa debe satisfacer una demanda actual diaria de 20,000 cajas.

La demanda durante este año fue un 10% inferior a la del año pasado, durante el año pasado la empresa realizó la producción en 40 lotes iguales a lo largo del año.

¿Cuánto dinero hubiera ahorrado SARDIMAR si hubiera empleado el modelo del lote económico de producción durante el año pasado? (Complete el siguiente cuadro) [Cuadro omitido por formato]

Durante este año, el área de producción está trabajando bajo el modelo EOQ. ¿Cuál será el nivel de inventarios en el momento que se emite cada orden de producción? Elabore la gráfica correspondiente a la situación descrita.

42.2 Datos identificados

Símbolo Significado Valor Unidad
\(p_{empaque}\) Capacidad de empaque (Cuello de botella) 750,000 cajas/mes
\(d_{actual}\) Demanda diaria actual 20,000 cajas/día
\(H\) Costo de mantener 10 soles/caja-año
\(S\) Costo de arranque (preparación) 2,000 soles/lote
\(LT\) Tiempo de entrega/arranque 4 días
\(N_{pasado}\) Lotes de producción año pasado 40 lotes/año

42.3 Supuestos de trabajo

  • Meses y Años: Se asume un año comercial estándar de 360 días y meses de 30 días, lo que es congruente con la relación de tasas diarias y mensuales en manufactura continua.
  • Cuello de Botella: Analizando el flujo secuencial, la etapa de “Empaque” limita el sistema a 750,000 cajas mensuales independientemente de las mermas anteriores, ya que las etapas previas superan esta capacidad neta. Por tanto, la tasa de producción \(p = 750,000 / 30 = 25,000 \text{ cajas/día}\).
  • Demanda del año pasado: La demanda actual es 10% inferior a la del año pasado. \(d_{actual} = 20,000 \implies d_{pasado} = \frac{20,000}{1 - 0.10} = \frac{20,000}{0.90} \approx 22,222.22 \text{ cajas/día}\). Anualizando: \(D_{pasado} = 22,222.22 \times 360 = 8,000,000 \text{ cajas/año}\). \(D_{actual} = 20,000 \times 360 = 7,200,000 \text{ cajas/año}\).
  • Modelo utilizado es EPQ (Economic Production Quantity) dado que hay una tasa de producción finita (\(p\)).

42.4 Desarrollo

42.4.1 a. Análisis del Año Pasado

1. Situación Real (40 lotes): \[ Q_{real} = \frac{D_{pasado}}{N} = \frac{8,000,000}{40} = 200,000 \text{ cajas/lote} \] Tasa de producción anual \(P = 25,000 \times 360 = 9,000,000 \text{ cajas/año}\). Factor EPQ \(= \left(1 - \frac{d_{pasado}}{p}\right) = \left(1 - \frac{22222.22}{25000}\right) = 1 - \frac{8}{9} = \frac{1}{9} \approx 0.1111\) \[ TC_S = N \times S = 40 \times 2,000 = \mathbf{80,000 \text{ soles}} \] \[ TC_H = \frac{Q}{2}\left(1 - \frac{d}{p}\right)H = \frac{200,000}{2} \left(\frac{1}{9}\right) 10 = 100,000 \times 1.1111 = \mathbf{1,111,111.11 \text{ soles}} \] \[ TC_{real} = 80,000 + 1,111,111.11 = \mathbf{1,191,111.11 \text{ soles}} \]

2. Situación Lote Económico (EPQ): \[ EPQ = \sqrt{\frac{2 D_{pasado} S}{H (1 - d/p)}} = \sqrt{\frac{2(8,000,000)(2000)}{10(1/9)}} = \sqrt{28,800,000,000} = \mathbf{169,705.63 \text{ cajas}} \] \[ TC_S(EPQ) = \left(\frac{8,000,000}{169,705.63}\right) 2000 = 47.1404 \times 2000 = \mathbf{94,280.90 \text{ soles}} \] \[ TC_H(EPQ) = \frac{169,705.63}{2} \left(\frac{1}{9}\right) 10 = \mathbf{94,280.90 \text{ soles}} \] \[ TC_{EPQ} = 94,280.90 + 94,280.90 = \mathbf{188,561.80 \text{ soles}} \]

Ahorro: \[ \text{Ahorro} = 1,191,111.11 - 188,561.80 = \mathbf{1,002,549.31 \text{ soles}} \]

Año pasado Cantidad a fabricar por lote (miles) Costo de Reposición (S/) Costo de Mantenimiento (S/) Costo Total (S/)
Situación real 200.00 80,000.00 1,111,111.11 1,191,111.11
Situación EPQ 169.71 94,280.90 94,280.90 188,561.80

42.4.2 b. Situación Este Año (Nivel de inventario al emitir orden)

Para este año, usando el modelo EPQ: \(d = 20,000\) cajas/día. \(p = 25,000\) cajas/día. Factor EPQ \(= \left(1 - \frac{20000}{25000}\right) = 1 - 0.8 = 0.2\) \[ D = 20,000 \times 360 = 7,200,000 \text{ cajas/año} \] \[ EPQ_{actual} = \sqrt{\frac{2(7,200,000)(2000)}{10(0.2)}} = \sqrt{14,400,000,000} = \mathbf{120,000 \text{ cajas}} \]

Tiempos del ciclo de producción: - Tiempo de ciclo (\(T\)) = \(EPQ / d = 120,000 / 20,000 = \mathbf{6 \text{ días}}\). - Tiempo de producción (\(t_1\)) = \(EPQ / p = 120,000 / 25,000 = \mathbf{4.8 \text{ días}}\). - Tiempo de consumo puro (\(t_2\)) = \(T - t_1 = 6 - 4.8 = \mathbf{1.2 \text{ días}}\). - Inventario Máximo (\(I_{max}\)) = \((p - d) \times t_1 = 5,000 \times 4.8 = \mathbf{24,000 \text{ cajas}}\).

Punto de pedido físico: Dado que el Lead Time (\(LT = 4 \text{ días}\)) es mayor que el tiempo de consumo puro (\(1.2 \text{ días}\)), la orden de producción se debe emitir mientras la máquina aún está produciendo el lote anterior. Si la orden se emite 4 días antes de que acabe el ciclo de 6 días, se emite en el día \(t = 6 - 4 = 2\) del ciclo. En el día 2, estamos en la fase de producción (que dura 4.8 días). El nivel físico de inventario en el día 2 es la acumulación neta de inventario: \[ Nivel_{fisico} = (p - d) \times 2 = (25,000 - 20,000) \times 2 = \mathbf{10,000 \text{ cajas}} \] (Nota: Si evaluamos la Posición de Inventario formal = Inventario físico + Órdenes en camino = \(ROP = d \times LT = 80,000\) cajas, pero físicamente en el almacén habrá 10,000 cajas).

Respuesta: El nivel de inventario físico en el momento de emitir la orden es de 10,000 cajas.

42.5 Implementación reproducible

Code 
# Parámetros Globales
dias_mes <- 30
meses_anio <- 12
dias_anio <- dias_mes * meses_anio

p_mensual <- 750000
p_diaria <- p_mensual / dias_mes
p_anual <- p_diaria * dias_anio

S <- 2000
H <- 10
LT <- 4

# a. AÑO PASADO
d_actual <- 20000
d_pasado <- d_actual / 0.90
D_pasado <- d_pasado * dias_anio

N_pasado <- 40
Q_real <- D_pasado / N_pasado

factor_epq_pasado <- (1 - d_pasado / p_diaria)

# Costos Reales
TCS_real <- N_pasado * S
TCH_real <- (Q_real / 2) * factor_epq_pasado * H
TC_real <- TCS_real + TCH_real

# Costos EPQ
EPQ_pasado <- sqrt((2 * D_pasado * S) / (H * factor_epq_pasado))
TCS_epq_pasado <- (D_pasado / EPQ_pasado) * S
TCH_epq_pasado <- (EPQ_pasado / 2) * factor_epq_pasado * H
TC_epq_pasado <- TCS_epq_pasado + TCH_epq_pasado

ahorro <- TC_real - TC_epq_pasado

cat("--- AÑO PASADO ---\n")
