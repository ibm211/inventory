70  Desarrollo del Caso 69

70.1 Enunciado literal

  1. EPQ y EOQ . “Calzado Peruviano S.A.C.” produce desde hace tres décadas una línea muy prestigiosa de botas de cuero para damas. Su proceso productivo incorpora también la producción del cuero, que es el proceso de curtiembre o conversión de piel vacuna en cuero (en este caso). El cuero se corta en las formas requeridas y, conjuntamente con los demás materiales, transita después por las fases de aparado, armado y acabado del producto final.

La demanda actual de botas es constante a lo largo del año, asciende a 5,000 pares de botas por mes; el precio de venta de este prestigioso calzado para dama es uno de los más altos del mercado: S/. 970 soles el par. La capacidad máxima de producción de estas botas asciende a 8,000 pares mensuales. Se trabajan 25 días al mes, los 12 meses del año.

Cada par de botas requiere 3 pies 2 de cuero; actualmente el costo de producción de un pie 2 de cuero en la curtiembre de “Calzado Peruviano S.A.C.”, es S/. 9.30 y la capacidad máxima de producción de la curtiembre, es de 60,000 pies 2 por trimestre. Adicionalmente se sabe que cada vez que la empresa tramita una orden de producción o de compra incurre en un costo de S/. 50 y el costo de iniciar la producción de un lote de cuero es S/. 250. También se conoce que para el costo de mantenimiento de los inventarios la empresa utiliza una tasa de transferencia mensual del 2.5%.

La preocupación respecto al insumo principal surge porque han recibido la propuesta de provisión del cuero por parte de Curtiembre ABC, a un precio de S/. 9.20 el pie 2, pero con la condición de proveer lotes de 30,000 pies2 en cada transacción. Adicionalmente, “Curtiembre Cuerex” ofrece la venta de cuero a un precio de S/. 9.25 el pie 2, pudiendo proveer la cantidad más conveniente que requiera la empresa en cualquier momento. El panorama actual requiere que alguien despeje todas las dudas y la confusión, y usted es la persona ideal para ello. En tal sentido, debe responder las siguientes preguntas:

  1. Si se produce ¿cuál es el tamaño ideal de los lotes de producción de cuero y el costo anual de producción? Sustente con los cálculos pertinentes.
  2. Qué es más conveniente ¿producir o comprar el cuero? Sustente rigurosamente su respuesta con los cálculos de tamaño de lote y de costos.

70.2 Datos identificados

Símbolo Significado Valor Unidad
Demanda Demanda anual de botas (\(5,000 \times 12\)) 60,000 pares/año
Req. Cuero requerido por par de botas 3 pies\(^2\)/par
\(D\) Demanda anual de cuero (\(60,000 \times 3\)) 180,000 pies\(^2\)/año
\(i\) Tasa de mantenimiento anual (\(2.5\% \times 12\)) 30% anual

70.2.1 Datos para Producción Interna (EPQ)

  • \(P_{cuero}\): Capacidad anual de curtiembre (\(60,000 \times 4\)) = \(240,000 \text{ pies}^2/\text{año}\).
  • \(C_{prod}\): Costo de producción = \(9.30 \text{ Soles/pie}^2\).
  • \(S_{prod}\): Costo total de preparación (\(50 + 250\)) = \(300 \text{ Soles/lote}\).
  • \(H_{prod}\): Costo de mantener (\(9.30 \times 0.30\)) = \(2.79 \text{ Soles/pie}^2\text{-año}\).

70.2.2 Datos para Compra (EOQ)

  • Costo base de trámite: \(S = 50 \text{ Soles/orden}\).

70.3 Desarrollo

70.3.1 a. Tamaño ideal de lotes de producción (Producir)

Cálculo del Lote Económico de Producción (EPQ): \[ Q_{epq}^* = \sqrt{\frac{2DS_{prod}}{H_{prod}\left(1 - \frac{D}{P}\right)}} \] \[ 1 - \frac{D}{P} = 1 - \frac{180000}{240000} = 1 - 0.75 = 0.25 \] \[ Q_{epq}^* = \sqrt{\frac{2(180000)(300)}{2.79(0.25)}} = \sqrt{\frac{108000000}{0.6975}} \approx \mathbf{12,443.42 \text{ pies}^2} \]

Costo Anual Total si se produce (\(TC_{prod}\)): \[ TC_{prod} = \left(\frac{D}{Q_{epq}^*}\right)S_{prod} + \left(\frac{Q_{epq}^*}{2}\right)H_{prod}(0.25) + (D \times C_{prod}) \] \[ TC_{prod} = 4339.64 + 4339.64 + 1674000 = \mathbf{\$1,682,679.28 \text{ anuales}} \]

70.3.2 b. ¿Qué es más conveniente: producir o comprar?

Evaluamos a los proveedores externos:

1. Proveedor Curtiembre ABC: - \(C = 9.20 \text{ Soles/pie}^2\). - Condición de lote: \(Q = 30,000 \text{ pies}^2\) (Fijo). - \(H = 9.20 \times 0.30 = 2.76 \text{ Soles/pie}^2\text{-año}\). \[ TC_{ABC} = \left(\frac{180000}{30000}\right) 50 + \left(\frac{30000}{2}\right) 2.76 + (180000 \times 9.20) \] \[ TC_{ABC} = 300 + 41400 + 1656000 = \mathbf{\$1,697,700.00 \text{ anuales}} \]

2. Proveedor Curtiembre Cuerex: - \(C = 9.25 \text{ Soles/pie}^2\). - Lote libre (aplicamos EOQ). - \(H = 9.25 \times 0.30 = 2.775 \text{ Soles/pie}^2\text{-año}\). \[ Q_{cuerex}^* = \sqrt{\frac{2(180000)(50)}{2.775}} \approx \mathbf{2,546.86 \text{ pies}^2} \] \[ TC_{cuerex} = \left(\frac{180000}{2546.86}\right) 50 + \left(\frac{2546.86}{2}\right) 2.775 + (180000 \times 9.25) \] \[ TC_{cuerex} = 3533.76 + 3533.77 + 1665000 = \mathbf{\$1,672,067.53 \text{ anuales}} \]

Conclusión b.: Comparando los tres escenarios: 1. Producción propia: $1,682,679.28 2. Curtiembre ABC: $1,697,700.00 3. Curtiembre Cuerex: $1,672,067.53

Es más conveniente COMPRAR a Curtiembre Cuerex en lotes de 2,546.86 pies\(^2\), ya que proporciona el menor costo total anual, generando un ahorro sustancial frente a la opción de producir o aceptar las condiciones rígidas del otro proveedor.

70.4 Implementación reproducible

Code 
# Variables Base
demanda_botas_anual <- 5000 * 12
cuero_por_bota <- 3
D <- demanda_botas_anual * cuero_por_bota
i <- 0.025 * 12

S_orden <- 50

# a. Producción propia (EPQ)
P_cuero <- 60000 * 4
C_prod <- 9.30
S_prod <- S_orden + 250
H_prod <- C_prod * i

factor_1_d_p <- 1 - (D / P_cuero)
Q_epq <- sqrt((2 * D * S_prod) / (H_prod * factor_1_d_p))
TC_prod <- (D / Q_epq)*S_prod + (Q_epq / 2)*H_prod*factor_1_d_p + (D * C_prod)

cat("--- A. PRODUCCIÓN PROPIA ---\n")
--- A. PRODUCCIÓN PROPIA ---
Code 
cat(sprintf("Lote óptimo a producir: %.2f pies cuadrados\n", Q_epq))
Lote óptimo a producir: 12443.42 pies cuadrados
Code 
cat(sprintf("Costo Total Anual: $%.2f\n\n", TC_prod))
Costo Total Anual: $1682679.29
Code 
# b. Opciones de Compra
# Proveedor ABC
C_abc <- 9.20
Q_abc <- 30000
H_abc <- C_abc * i
TC_abc <- (D / Q_abc)*S_orden + (Q_abc / 2)*H_abc + (D * C_abc)

# Proveedor Cuerex
C_cuerex <- 9.25
H_cuerex <- C_cuerex * i
Q_cuerex <- sqrt((2 * D * S_orden) / H_cuerex)
TC_cuerex <- (D / Q_cuerex)*S_orden + (Q_cuerex / 2)*H_cuerex + (D * C_cuerex)

cat("--- B. OPCIONES DE COMPRA ---\n")

cat(sprintf("Proveedor ABC (Lote fijo %d): TC = $%.2f\n", Q_abc, TC_abc))

cat(sprintf("Proveedor Cuerex (Lote EOQ %.2f): TC = $%.2f\n\n", Q_cuerex, TC_cuerex))

Code 
# Comparativa
opciones <- c(Produccion = TC_prod, ABC = TC_abc, Cuerex = TC_cuerex)
mejor_opcion <- names(which.min(opciones))
