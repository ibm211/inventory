3 Caso 1 – EOQ: Lote Económico de Compra (Empresa XYZ)

3.1 Enunciado

  1. EOQ. El Gerente de Operaciones de la empresa “XYZ” desea determinar las unidades que deberían comprarse de ciertos productos para minimizar los costos. La información que le proporcionaron fue la siguiente:
  • Demanda anual, en unidades: 10,000
  • Costo del producto ($/unidad): 0.80
  • Días laborales al año: 250
  • Costo administrativo por ordenar un pedido ($/lote): 30
  • Costo de capital mensual: 1.5%
  • Costo de obsolescencia por año: 1.0%
  • Costo de almacenamiento anual: 1.0%
  1. Determinar el tamaño de lote económico. Rpta.: 1,936.5 unidades/pedido
  2. Calcular el número esperado de pedidos. Rpta.: 5.16 pedidos/año
  3. Calcular el tiempo esperado entre pedidos. Rpta.: 2.33 meses
  4. Calcular el costo total anual de inventario.
  5. Graficar IOH vs. Tiempo.

3.2 Datos del problema

Variable Símbolo Valor Unidad
Demanda promedio D 10,000 unidades/año
Costo unitario de compra/producción c 0.80 $/unidad
Días laborales al año — 250 días/año
Costo fijo de ordenar K 30 $/pedido
Costo de capital mensual i_k 1.5% mensual
Costo de obsolescencia i_o 1.0% anual
Costo de almacenamiento i_a 1.0% anual

3.3 Modelo aplicable

Modelo EOQ (Economic Order Quantity) — Lote Económico de Compra con reposición instantánea.

3.3.1 Cálculo del costo de mantener inventario (h)

El costo de mantener una unidad en inventario por año se compone de tres factores aplicados sobre el costo unitario de compra/producción (c): h = c \times (i_k \times 12 + i_o + i_a) h = 0.80 \times (0.015 \times 12 + 0.01 + 0.01) = 0.80 \times (0.18 + 0.01 + 0.01) = 0.80 \times 0.20 = 0.16 \text{ \$/unidad-año}

3.4 Desarrollo

3.4.1 Resolución Matemática en LaTeX

a) Tamaño de lote económico (Q^*) Q^* = \sqrt{\frac{2DK}{h}} = \sqrt{\frac{2 \times 10{,}000 \times 30}{0.16}} = \sqrt{\frac{600{,}000}{0.16}} = \sqrt{3{,}750{,}000} \approx 1936.49 \text{ unidades/pedido}

b) Frecuencia de pedidos (F) F = \frac{D}{Q^*} = \frac{10{,}000}{1{,}936.49} \approx 5.164 \text{ pedidos/año}

c) Tiempo de ciclo entre pedidos (T) En meses (12 meses por año): T = \frac{12}{F} = \frac{12}{5.164} \approx 2.324 \text{ meses} En días laborables: T_{días} = \frac{250}{F} = \frac{250}{5.164} \approx 48.42 \text{ días laborables}

d) Costo total relevante anual (TRC) y Costo total anual (TC) - Costo total relevante anual (TRC): incluye únicamente los costos que dependen del tamaño del lote (ordenar y mantener). TRC = \frac{D}{Q^*}K + \frac{Q^*}{2}h TRC = \left(\frac{10{,}000}{1{,}936.49}\right)(30) + \left(\frac{1{,}936.49}{2}\right)(0.16) = 154.92 + 154.92 = 309.84 \text{ \$/año}

  • Costo total anual (TC): incluye el costo de compra/obtención de las unidades más el costo relevante de inventario. TC = cD + TRC TC = 0.80 \times 10{,}000 + 309.84 = 8{,}000 + 309.84 = 8{,}309.84 \text{ \$/año}

3.5 Implementación en R

Code 
# Parámetros del problema
D <- 10000
c <- 0.80
dias_lab <- 250
K <- 30
i_capital <- 0.015
i_obsol <- 0.01
i_almac <- 0.01

# Cálculo de h
tasa_total <- i_capital * 12 + i_obsol + i_almac
h <- c * tasa_total

# a) Lote económico Q*
Q_star <- sqrt(2 * D * K / h)
cat(sprintf("a) Q* = %.2f unidades/pedido\n", Q_star))
a) Q* = 1936.49 unidades/pedido
Code 
# b) Frecuencia de pedidos (F)
F_pedidos <- D / Q_star
cat(sprintf("b) F = %.4f pedidos/año\n", F_pedidos))
b) F = 5.1640 pedidos/año
Code 
# c) Tiempo de ciclo entre pedidos (T)
T_meses <- 12 / F_pedidos
T_dias <- dias_lab / F_pedidos
cat(sprintf("c) T = %.2f meses o %.2f días laborables\n", T_meses, T_dias))
c) T = 2.32 meses o 48.41 días laborables
Code 
# d) Costos
costo_ordenar <- (D / Q_star) * K
costo_mantener <- (Q_star / 2) * h
TRC <- costo_ordenar + costo_mantener
TC <- c * D + TRC
cat(sprintf("d) Costo Total Relevante (TRC) = $%.2f/año\n", TRC))
d) Costo Total Relevante (TRC) = $309.84/año
Code 
cat(sprintf("   Costo Total Anual (TC, incluye compra) = $%.2f/año\n", TC))
   Costo Total Anual (TC, incluye compra) = $8309.84/año
Code 
# e) Gráfica IOH vs. Tiempo
d_diaria <- D / dias_lab
ciclos <- 3
t_total <- c()
ioh_total <- c()
t_acum <- 0

for (i in 1:ciclos) {
  t_inicio <- t_acum
  t_fin <- t_acum + T_dias
  t_ciclo <- seq(t_inicio, t_fin, length.out=100)
  ioh_ciclo <- Q_star - d_diaria * (t_ciclo - t_inicio)
  
  t_total <- c(t_total, t_ciclo)
  ioh_total <- c(ioh_total, ioh_ciclo)
  t_acum <- t_fin
}

plot(t_total, ioh_total, type="l", col="blue", lwd=2,
     xlab="Tiempo (días laborables)", ylab="Inventario On Hand (IOH)",
     main="Caso 1 - Gráfica IOH vs. Tiempo", ylim=c(0, max(ioh_total)*1.1))
abline(h = Q_star/2, col="red", lty=2)
grid()

3.6 Resumen de resultados

Concepto Símbolo Valor Unidad
Costo de mantener inventario h 0.16 $/unidad-año
Lote económico Q^* 1,936.49 unidades/pedido
Frecuencia de pedidos F 5.16 pedidos/año
Tiempo de ciclo (meses) T 2.33 meses
Tiempo de ciclo (días laborables) T_{\text{días}} 48.42 días laborables
Costo anual de ordenar \frac{D}{Q^*}K 154.92 $/año
Costo anual de mantener \frac{Q^*}{2}h 154.92 $/año
Costo total relevante (TRC) TRC 309.84 $/año
Costo total anual (TC) TC 8,309.84 $/año

3.7 Conclusiones

  1. Lote económico de 1,936.5 unidades: Este tamaño de pedido equilibra los costos de ordenar y mantener inventario, resultando en el mínimo costo total relevante (TRC).
  2. Igualdad de costos en el óptimo: En el punto EOQ, el costo anual de ordenar ($154.92) es exactamente igual al costo anual de mantener ($154.92). Esta es una propiedad fundamental del modelo EOQ que sirve como verificación del resultado.
  3. Frecuencia moderada: Con 5.16 pedidos al año (aproximadamente cada 2.33 meses o 48 días laborables), la empresa mantiene un balance adecuado entre la frecuencia de pedidos y el nivel de inventario.
  4. Costo total relevante vs. Costo total anual: Es crucial distinguir entre el costo total relevante (TRC), que es de $309.84/año y sirve para optimizar el inventario, y el costo total anual (TC), que incluye la compra/adquisición de los productos (c \times D = \$8,000.00) sumando un total de $8,309.84/año.
  5. Composición del costo de mantener: El componente principal de h es el costo de capital, que representa la mayor parte de la tasa total de mantenimiento (20%). representa la mayor parte de la tasa total de mantenimiento (20%).
