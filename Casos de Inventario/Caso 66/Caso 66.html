67  Desarrollo del Caso 66

67.1 Enunciado literal

  1. EPQ. La empresa Dulces y Sabores se dedica a la elaboración de chocolates y tiene como mercado el sur del país. Actualmente tiene una capacidad instalada de 40,000 unidades anuales. La demanda es constante y del orden de 3,000 unidades mensuales. Para preparar una orden de compra o de producción se requieren 15 soles para gastos administrativos; también se deben considerar 60 soles para preparar el equipo por cada lote que se va a producir.

Almacenar una unidad durante un mes cuesta 5 soles y se cuenta con un stock de seguridad permanente de 50 unidades. Se trabajan 250 días al año y los 12 meses sin interrupción.

  1. Calcule el lote económico de producción y el costo total de inventario para la empresa Dulces y Sabores.
  2. ¿Cuál será la cantidad existente en el almacén 5 días después de hacer empezado un nuevo ciclo de producción? Represente su respuesta en un gráfico Cantidad de Inventario vs Tiempo.

67.2 Datos identificados

Símbolo Significado Valor Unidad
\(P\) Capacidad instalada de producción 40,000 unidades/año
\(D_{mes}\) Demanda mensual 3,000 unidades/mes
\(D\) Demanda anual (\(3,000 \times 12\)) 36,000 unidades/año
\(S\) Costo de ordenar/preparar (\(15 + 60\)) 75 Soles/lote
\(H_{mes}\) Costo de mantener mensual 5 Soles/unidad-mes
\(H\) Costo de mantener anual (\(5 \times 12\)) 60 Soles/unidad-año
\(SS\) Stock de seguridad 50 unidades
\(días\) Días laborables por año 250 días/año

67.3 Desarrollo

67.3.1 a. Lote económico de producción y Costo Total

Cálculo del Lote Económico de Producción (EPQ): \[ Q_{epq}^* = \sqrt{\frac{2DS}{H\left(1 - \frac{D}{P}\right)}} \] \[ 1 - \frac{D}{P} = 1 - \frac{36000}{40000} = 1 - 0.9 = 0.1 \] \[ Q_{epq}^* = \sqrt{\frac{2(36000)(75)}{60(0.1)}} = \sqrt{\frac{5400000}{6}} = \sqrt{900000} \approx \mathbf{948.68 \text{ unidades}} \]

Costo Total del Inventario (\(TC_{gestion}\)): Calculamos el Inventario Máximo Cíclico: \[ I_{max\_ciclico} = Q_{epq}^*\left(1 - \frac{D}{P}\right) = 948.68 \times 0.1 = 94.87 \text{ unidades} \] \[ TC_{gestion} = \left(\frac{36000}{948.68}\right) 75 + \left(\frac{94.87}{2}\right) 60 + (50 \times 60) \] \[ TC_{gestion} = 2846.05 + 2846.10 + 3000 = \mathbf{\$8,692.15 \text{ anuales}} \]

67.3.2 b. Inventario a los 5 días de iniciado el ciclo

Al inicio de un ciclo, el inventario es igual al Stock de Seguridad (\(SS = 50\)). Durante la fase de producción, el inventario crece a una tasa neta diaria de \((p - d)\). \[ p = \frac{40000}{250} = 160 \text{ unidades/día} \] \[ d = \frac{36000}{250} = 144 \text{ unidades/día} \] Tasa de crecimiento = \(160 - 144 = 16 \text{ unidades/día}\).

Inventario a los 5 días (\(I_{5d}\)): \[ I_{5d} = SS + (\text{Tasa de crecimiento} \times 5) = 50 + (16 \times 5) \] \[ I_{5d} = 50 + 80 = \mathbf{130 \text{ unidades}} \]

67.4 Implementación reproducible

Code 
# Variables Base
P <- 40000
D <- 3000 * 12
S <- 15 + 60
H <- 5 * 12
SS <- 50
dias_anio <- 250

# a. Lote económico (EPQ) y Costo Total
factor_1_d_p <- 1 - (D / P)
Q_epq <- sqrt((2 * D * S) / (H * factor_1_d_p))

I_max_ciclico <- Q_epq * factor_1_d_p
C_o <- (D / Q_epq) * S
C_m_ciclo <- (I_max_ciclico / 2) * H
C_m_ss <- SS * H
TC_gestion <- C_o + C_m_ciclo + C_m_ss

cat("--- A. EPQ Y COSTOS ---\n")
--- A. EPQ Y COSTOS ---

Code

# b. Inventario a los 5 días
p <- P / dias_anio
d <- D / dias_anio
tasa_crecimiento <- p - d
inventario_5_dias <- SS + (tasa_crecimiento * 5)
