41  Desarrollo del Caso 40

41.1 Enunciado literal

  1. Un Haggis es la carne de un platillo escocés. La compañía “Rex” tiene una máquina de Haggis que puede producir a una tasa de 1,000 unidades diarias. Por tradición, la compañía produce todo un día y cubre la demanda de una semana. Cada vez que se prepara la máquina para producir un lote se gasta $ 100 y se demora medio día. Los productos se almacenan en condiciones especiales de frío e higiene y le cuesta a la compañía 0.5 $ por unidad al día. La empresa trabaja cinco días por semana, 50 semanas al año.
  1. ¿Cuánto dinero pierde la compañía anualmente por producir 1,000 unidades a la semana en un solo lote en lugar de trabajar con el modelo de Lote económico? ¿Cuál es el motivo principal de la diferencia?
  2. ¿Cómo se vería afectado el punto de pedido actual si se decide que a partir del próximo mes se va a trabajar basándose en el modelo de Lote Económico? Sustente su respuesta con el gráfico respectivo.

41.2 Datos identificados

Símbolo Significado Valor Unidad
\(p\) Tasa de producción diaria 1,000 unid/día
Días Días laborales por semana 5 días/sem
Semanas Semanas laborales por año 50 sem/año
\(S\) Costo de preparación de máquina 100 $/orden
\(LT\) Tiempo de preparación (demora) 0.5 días
\(H_{diario}\) Costo de mantenimiento diario 0.5 $/unid-día
\(Q_{actual}\) Lote de producción actual 1,000 unidades

41.3 Supuestos de trabajo

  • Días laborables al año = \(5 \times 50 = 250 \text{ días/año}\).
  • La política actual indica que “produce todo un día y cubre la demanda de una semana”. Si produce a 1,000 unid/día durante 1 día, el lote actual es de \(1,000 \text{ unidades}\). Si eso cubre la demanda de 1 semana, la demanda semanal es de \(1,000 \text{ unidades}\).
  • Demanda anual (\(D\)) = \(1,000 \text{ unid/sem} \times 50 \text{ sem/año} = 50,000 \text{ unid/año}\).
  • Demanda diaria (\(d\)) = \(1,000 / 5 = 200 \text{ unid/día}\).
  • Costo anual de mantenimiento (\(H\)) = \(0.5 \text{ \$/unid-día} \times 250 \text{ días/año} = 125 \text{ \$/unid-año}\).

41.4 Desarrollo

41.4.1 a. Pérdida anual por no usar el Lote Económico (EPQ)

Este es un modelo de inventarios con producción gradual (EPQ).

1. Costo con la política actual (\(Q = 1,000\) unidades): \[ TC_S = \left(\frac{D}{Q}\right) S = \left(\frac{50000}{1000}\right) 100 = 50 \times 100 = \mathbf{5,000 \text{ \$/año}} \] \[ TC_H = \frac{Q}{2}\left(1 - \frac{d}{p}\right)H = \frac{1000}{2}\left(1 - \frac{200}{1000}\right) 125 = 500(0.8)125 = 400 \times 125 = \mathbf{50,000 \text{ \$/año}} \] \[ TC_{actual} = 5,000 + 50,000 = \mathbf{55,000 \text{ \$/año}} \]

2. Costo con el Lote Económico de Producción (EPQ): \[ EPQ = \sqrt{\frac{2DS}{H\left(1 - \frac{d}{p}\right)}} = \sqrt{\frac{2(50000)(100)}{125(0.8)}} = \sqrt{\frac{10,000,000}{100}} = \sqrt{100,000} \approx \mathbf{316.23 \text{ unidades}} \] \[ TC_{S\_EPQ} = \left(\frac{50000}{316.23}\right) 100 = 158.11 \times 100 = 15,811.39 \text{ \$/año} \] \[ TC_{H\_EPQ} = \frac{316.23}{2}(0.8)125 = 158.115 \times 100 = 15,811.39 \text{ \$/año} \] \[ TC_{EPQ} = 15,811.39 + 15,811.39 = \mathbf{31,622.78 \text{ \$/año}} \]

Dinero perdido anualmente: \[ \text{Pérdida} = TC_{actual} - TC_{EPQ} = 55,000 - 31,622.78 = \mathbf{23,377.22 \text{ \$/año}} \]

Motivo de la diferencia: La política actual de producir un lote masivo para toda la semana infla drásticamente el inventario promedio almacenado. Como el costo de mantenimiento físico (especialmente frío e higiene) es altísimo (0.5$ diarios), mantener inventario ocioso resulta extremadamente costoso frente al ahorro en costos de preparación.

41.4.2 b. Efecto sobre el Punto de Pedido (ROP)

El Punto de Pedido se define exclusivamente por la demanda durante el tiempo de espera (Lead Time), independientemente de la política de tamaño de lote elegida. \[ ROP = d \times LT = 200 \text{ unid/día} \times 0.5 \text{ días} = \mathbf{100 \text{ unidades}} \]

Respuesta: El punto de pedido NO se vería afectado en lo absoluto. En ambos modelos (Actual y EPQ), la máquina debe comenzar a prepararse (demorando medio día) justo cuando el inventario cae a 100 unidades, para que la producción inicie exactamente cuando el inventario llegue a 0.

41.5 Implementación reproducible

Code 
# Variables Base
p_diaria <- 1000
dias_semana <- 5
semanas_anio <- 50
S <- 100
LT <- 0.5
H_diario <- 0.5
Q_actual <- 1000

# Cálculos derivados
dias_anio <- dias_semana * semanas_anio
d_diaria <- Q_actual / dias_semana # Demanda de 1 semana cubierta por el lote
D_anual <- d_diaria * dias_anio
H_anual <- H_diario * dias_anio
factor_epq <- (1 - d_diaria / p_diaria)

# a. Análisis de Costos (Actual vs EPQ)
# Costo Actual
TCS_actual <- (D_anual / Q_actual) * S
TCH_actual <- (Q_actual / 2) * factor_epq * H_anual
TC_actual <- TCS_actual + TCH_actual

# Costo EPQ
EPQ <- sqrt((2 * D_anual * S) / (H_anual * factor_epq))
TCS_epq <- (D_anual / EPQ) * S
TCH_epq <- (EPQ / 2) * factor_epq * H_anual
TC_epq <- TCS_epq + TCH_epq

perdida <- TC_actual - TC_epq

cat("a. Análisis de Pérdida Financiera:\n")
